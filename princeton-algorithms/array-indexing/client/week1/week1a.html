How to compile or run java program?

Write code in java.
- Compile it using javac HelloWorld.java and run the java code by java HelloWorld

Breakdown of public static void main(String[] args)

public - This is an access modifier, which means that the main method is accessible from anywhere. It needs to be public so that the Java Virtual Machine (JVM) can call this method to start the program.

static - keyword means method belongs to class instead of instance of class as this is starting point so called without creating objects

void - main method doesnt return any value

main - name of method: JVM looks for main method when it starts executing the java program. Cannot change the name of method

String[] args - This is the parameter to the main method. It is an array of String objects, and it stores any command-line arguments that are passed to the program. Can change name of "args". Below args array will contain {"arg1", "arg2"} 

        java MyProgram arg1 arg2

Union-Find algorithm

Union - connect two objects
Find/connected query - is there a path connecting the two objects?

Assumptions:

  • Reflexive: p is connected to p
  • Symmetric: if p is connected to q, then q is connected to p
  • Transitive: if p is connected to q and q is connected to r, then p is connected to r

Union find data type api:

        UF(int n); //initialize union find datastructure with n objects(0-N-1)
        void union(int p, int q) //add connection between p and q
        boolean connected(int p, int q) // are p and q in same component?
        int find(int p)
        int count()
