What is Collection?

This is group of objects or a single entity representating multiple objects

In java to represent group of objects in single entity we need classess and interfaces. Collection framework provides these.

  • Example: 3 students (Stu1, stud2, stu3) = collection. To represent group of element we need collection:
  • Ex: ArrayList: In this we can store group of elements
  • Collection framework are group of classes and interfaces used to store the group of objects like students
  • Object[] a = new Object[]; // This can hold homogenous or heterogenous
  • To overcome some known limitations like homogeous, fixed size, no api's we have Collection frameworks

Difference between Collection and Collections

  • Collection is an interface defined in java.util package Example are:
    • List, Set, Hasmap etc
  • Collections is a utility class defined in util package it provides static methods which operate on collections 
    •         Collection <String> numbers = new ArrayList<>();
        numbers.add("1");
        numbers.add("2");
        Collections.sort(numbers); // Here we are using specific method "sort" from collections
        System.out.println(numbers);

What is Iterable?

Iterable is interface present in java.lang.iterable package. It allows users to iterate through elements from collection. Three ways to iterate through :

  • Enhanced for loop
  • forEach loop
  • Iterator() method
Collection extends Iterable.

It represents a data structure that can be iterated over.The Iterable interface provides a method that produces an Iterator When using an Iterable, we cannot get an element by index. Similarly, we cannot get the first or the last elements from the data structure as well.


Iterable is interface in java which allows collection element to be accessed individually. Arrays, queue, sets can be traversed easily

Any class that implements an Iterable interface overrides the iterator() method present in the Iterable interface.
iterator() method calls an iterator interface which returns iterator. Methods of Iterator interface is next(), hasNext(), remove() and forEachRemaining().
