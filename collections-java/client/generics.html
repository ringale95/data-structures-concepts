Why generics?

Developers have to make new copy of class if we want to change the type of a variable or method. So code duplication would come. That's why we have generics

What is Generics?

Generics in Java is a language feature introduced in JDK 5 that allows you to define classes, interfaces, and methods with a placeholder for the types they operate on. This means you can write code that can operate on objects of various types while providing compile-time type safety.
Generics dont work with primitive types use Wrapper class which is "Integer".

What is wrapper class?

Wrapper classes are set of classes in java.lang package that wrap primitive data types into objects
It has Conversion methods and constants(INTEGER.MIN_VALUE)
