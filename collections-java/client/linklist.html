What is LinkList?

A linked list is a fundamental data structure in computer science that consists of a sequence of elements, called nodes, where each node contains a value and a reference (or link) to the next node in the sequence.

We use link list:
Dynamic size
Insertion and deletion

Types of Linked list are:

  • Singly link list
    • Disadvantage: Need reference and it complete traversal to be done: Adding/Deleting element O(n)
    • Reverse traversal not available
    • All operation start with HEAD
    • Some operations are hard to perform
  • Doubly link list
    • Advantage: Traversal is easy due to two pointers. At any pointer u can traversal
  • Circular link list

Operations on link list:

  • Traversal
  • Insertion
  • Deletion
  • Search
  • Concatenation

Linklist is implementation of List interface. Linklist is dynamic allocation


In collection API we have linklist operations: LinkedList list = new Linkedlist<>();
list.add(5);
